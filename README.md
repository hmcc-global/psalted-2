# Psalted 2 Repo

A repo built on ExpressJS and React with TypeScript

## Installation

1. Clone the repo
2. Run `yarn install`
3. Run `cd server && yarn install`
4. Run `cd ui && yarn install`

## Usage

1. Open your choice of Terminal/cmd/PowerShell
2. Run `cd server && yarn dev`
